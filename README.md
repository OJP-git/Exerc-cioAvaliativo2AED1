##LeetCode 503: Next Greater Element II
###https://leetcode.com/problems/next-greater-element-ii/description/
Esse leetcode fornecia um array circular para que fosse encontrado o próximo maior elemento dentro dele. Considerando do numero até o final e do inicio do array até o próprio numero. Caso não fosse encontrado, a função retornaria -1 no lugar de um numero do array fornecido.
###Casos testados:
####Input: [5,4,3,2,1], output: [-1,5,5,5,5]
####Input: [1,2,1], output: [2,-1,2]
####Input: [3,5,4,3,3,1,5], output: [5,-1,5,5,5,5,-1]
